# Timer App

A timer application with Cypress end-to-end testing.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
```bash
npm install
```

### Running the Application
```bash
npm start
```
The app will be available at http://localhost:8000

### Running Tests
```bash
# Open Cypress test runner
npm run cypress:open

# Run tests headlessly
npm run cypress:run

# Start app and run tests
npm run test
```

## Scripts

- `npm start` - Start the timer application
- `npm run cypress:open` - Open Cypress test runner
- `npm run cypress:run` - Run Cypress tests headlessly
- `npm run test` - Start app and run tests

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Cypress (E2E Testing)
- Serve (Development server)
